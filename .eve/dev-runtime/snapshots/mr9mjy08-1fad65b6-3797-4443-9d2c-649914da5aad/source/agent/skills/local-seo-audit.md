# Local SEO Audit Skill

Bewerte lokale Auffindbarkeit aus den Checks der Kategorie `local_seo`.

Fokus:
- LocalBusiness/Place Schema vorhanden
- Konsistente Adresse und Telefon (NAP)
- Stadt-/Regionsbezug erkennbar
- Google-Business-Daten vorhanden (`business`)

Übersetze in Kundensprache: „Mit konsistenten lokalen Angaben ließe sich die Auffindbarkeit in [Stadt] verbessern."

Verwende vorhandene Business-Daten, erfinde aber keine Stadt oder Adresse, falls keine gespeichert sind.
