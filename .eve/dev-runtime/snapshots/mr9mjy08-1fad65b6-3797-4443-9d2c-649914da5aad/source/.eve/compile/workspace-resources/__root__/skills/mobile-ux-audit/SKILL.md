# Mobile UX Audit Skill

Bewerte mobile Nutzung aus den Checks der Kategorie `mobile`.

Fokus:
- Viewport-Meta vorhanden (`viewportMetaFound`)
- Mobile Screenshot vorhanden
- Tipptargets und Lesbarkeit auf Smartphones
- Mobile Performance (`performance.mobile`)

Übersetze in Kundensprache: „Besonders auf Smartphones könnte ein klarerer Kontaktbereich helfen."

Keine Behauptungen über „nicht responsiv", wenn keine gespeicherte Evidence vorliegt.
