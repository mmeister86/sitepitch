# SEO Basics Audit Skill

Bewerte SEO-Grundlagen aus den Checks der Kategorie `seo`.

Fokus:
- Title und Meta Description vorhanden und aussagekräftig
- Canonical URL korrekt
- robots.txt und Sitemap gefunden
- Strukturierte Daten (`schemaTypes`)
- H1/H2-Hierarchie

Übersetze in Kundensprache: „Die lokale Auffindbarkeit ließe sich mit wenigen strukturellen Anpassungen verbessern."

Evidence nur aus gespeicherten Checks. Keine Ranking-Garantien.
